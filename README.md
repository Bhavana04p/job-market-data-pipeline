# Job Market Data Pipeline

A Data Engineering project built using Apache Airflow, Docker, PostgreSQL, and Python.

## Project Flow

Jobs API → Airflow ETL → PostgreSQL

## Tech Stack

- Apache Airflow
- Docker
- PostgreSQL
- Python
- Requests

## Features

- API Data Extraction
- Data Transformation
- PostgreSQL Loading
- Workflow Orchestration
- XCom Data Sharing
